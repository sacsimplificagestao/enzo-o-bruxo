Página de links oficial do **Enzo O Bruxo** — visual futurista, tema neon verde (lançamento Brega do Bruxo 3.0), otimizada para celular e desktop (layout centralizado tipo "celular" também no PC).

## Stack
HTML + CSS + JS puro, sem dependências. Fontes: Orbitron + Rajdhani (Google Fonts).

## Rodar localmente
```bash
python3 -m http.server 5500
```
Depois acesse: http://localhost:5500

## Estrutura
```
index.html            # página única (HTML + CSS + JS)
assets/
  logo.png            # logo branca
  avatar.jpg          # foto de destaque
  enzo.jpg            # foto alternativa (retrato)
```
